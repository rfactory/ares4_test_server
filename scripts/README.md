# 프로덕션 스크립트

이 디렉터리에는 프로덕션 환경에서 직접 실행되거나 컨테이너의 진입점(entrypoint)으로 사용되는 스크립트 파일들만 포함되어야 합니다.

## 원칙

- **단일 책임:** 각 스크립트는 하나의 명확한 책임을 가져야 합니다. (예: 데이터베이스 마이그레이션, 특정 백그라운드 작업 실행)
- **환경 변수 사용:** 스크립트 내에 설정 값(예: 호스트 주소, 비밀번호)을 하드코딩하는 것을 절대 금지합니다. 모든 설정은 환경 변수를 통해 주입받아야 합니다.
- **테스트 코드 금지:** 이 디렉터리에는 **절대** 테스트 코드, 임시 테스트 스크립트, 혹은 개발 중에만 사용하는 코드를 포함해서는 안 됩니다. 테스트 코드는 `tests/` 디렉터리에 위치해야 합니다.
